# Week 10 Journal - Vivian

## Overview
This week’s efforts focused on reconciling legacy frontend components with our new design system and establishing a cohesive brand identity for the application through custom design assets.

## Learning Objectives Met

### 1. Implement frontend and design frontend architecture
* **UI Standardization & Theme Integration:** I refactored the "Checker Mode" page to align with our established frontend architecture. This involved migrating to a modular directory structure and replacing hard-coded styles with MUI’s theme system. By adding specific color tokens (warning, success, info) to our global theme, I’ve ensured that the interface remains consistent and easily maintainable as the project scales.
* **Dependency Optimization:** I replaced custom SVG assets with standardized MUI icons. This not only ensures visual consistency across the platform but also reduces bundle overhead, adhering to best practices for frontend performance.

### 2. Master high-fidelity prototyping and design tooling
* **Logo Design & Integration:** I utilized Figma to design the project’s favicon, creating multiple iterations to test visibility across different browser themes. I implemented the dark version of the logo while establishing a design roadmap to transition toward the lighter Clean Architecture color palette.
